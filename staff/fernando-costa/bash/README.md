# bash

Command-line interface commands typical in BASH.

## pwd
pwd= path work directory => qué carpeta estamos
```bash
$ pwd
```

## ls
todas las carpetas
```bash
$ ls
```

## ls -l
todas las carpetas pero más info
```bssh
$ ls -l
```

## touch

```bash
$ touch
```

## ls -a

## mkdir (make directory)
